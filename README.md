## Compilation
Pour utiliser le `Makefile` fourni, utilisez la commande suivante :

```bash
make
```

## Utilisation
Pour lancer le programme, utilisez la commande suivante :

```bash
./flappy
```

## Comment jouer
* `Espace` : sauter
* Passez entre les tuyaux pour gagner des points. Si vous touchez un tuyau, vous perdez.
